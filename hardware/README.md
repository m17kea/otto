# Hardware

CAD files, wiring diagrams, and parts lists for the Otto DIY robot.

For the full public build guide, use [`docs/BUILD.md`](../docs/BUILD.md).
That guide is the canonical place for:
- the recommended classroom build
- direct STL links
- required and optional components
- example Amazon UK and AliExpress buying links

## Printables STL files
- `hardware/printables/otto_diy/` contains the STL files used for this build.
- `hardware/printables/otto_diy_humanoid/` contains the humanoid expansion STL files.
