# Security Policy

## Supported Versions

Security fixes are provided on a best-effort basis for the default branch.

## Reporting a Vulnerability

Please do **not** report security vulnerabilities in public issues.

Use GitHub private vulnerability reporting for this repository:

- https://github.com/m17kea/otto/security/advisories/new

If private reporting is unavailable, open a normal issue asking for a private contact channel without sharing sensitive details.
